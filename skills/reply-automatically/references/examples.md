# Draft Reply — 出力例

判定結果や前置きを出さず、返信文のみを出力する点に注意する。

## 例1：社外・メール・日本語（依頼への返答）

**入力コンテキスト（アクティブウィンドウ）**
> 差出人：田中（△△製薬）
> お世話になっております。先日ご提案いただいた件、社内で検討したく資料を拝見できればと存じます。ご送付いただけますでしょうか。

**良い出力**
```
田中さま

お世話になっております。ご検討いただき、誠にありがとうございます。

ご提案資料を添付いたしますので、ご高覧いただけますと幸いです。ご不明な点がございましたら、お気軽にお申し付けください。

ご検討のほど、何卒よろしくお願いいたします。
```

**NG出力（前置き・解説が入っている）**
```
これは社外向けのビジネスメールですね。丁寧なトーンで返信文を作成しました。

田中さま
お世話になっております……
```

## 例2：社内・Slack・日本語（確認）

**入力コンテキスト**
> 明日の定例、10時開始で認識合ってる？アジェンダ共有しておいてほしい。

**良い出力**
```
10時開始で合ってます。アジェンダは今日中に共有します。
```

**NG出力（過剰な敬語・冗長）**
```
お疲れ様です。ご確認いただきありがとうございます。明日の定例につきましては10時開始で相違ございません。アジェンダにつきましては本日中に共有させていただきます。
```

## 例3：社外・メール・英語（回答）

**入力コンテキスト**
> From: John (Acme Corp)
> Hi, could you let us know your availability for a call next week?

**良い出力**
```
Hi John,

Thank you for reaching out. I'd be happy to set up a call next week.

I'm available on [Tuesday and Thursday afternoon]. Please let me know what works best on your side, and I'll send an invite.

Best regards,
```

## 例4：情報不足のケース

**入力コンテキスト**
> （空、または返信対象が判別できない画面）

**良い出力**
```
返信対象の内容が見つかりません
```
